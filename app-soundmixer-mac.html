Sound Mixer - macOS

Create an ordered list of sound files to play individually or simultaneously

Key Features

Create Sections

Create sections to label groups of sound files.

Stop All Other Sounds

Ability to stop all currently playing sounds when another sound file is played.

Loop

Play specific sounds in a loop.

Lock

Lock to prevent accidental changes to the sound file list.

Notes

Add notes to a sound file.

Shortcut Keys

Set specific sounds to play when a key is pressed (when the app has focus), or press Return to play the currently selected sound file.
