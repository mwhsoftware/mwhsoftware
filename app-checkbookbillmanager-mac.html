Checkbook Bill Manager - macOS

A simple checkbook and bill manager to help organize your finances

Key Features

Accounts

Keep track of any number of accounts.

Transactions

For each transaction, track the date, name, type, amount, check number, category, and notes.

Filter

Ability to filter transactions displayed.

Charts

View charts for Income or Expenses by category for a specified date range.

Mark Bill Paid

When adding a transaction, quickly mark a bill as paid. This will set the payment date and amount for the payment.

Add a Bill

Specify the name, due date, amount, and repeat type of the bill.

Mark Paid

Keep track of the amount and date paid.

View Payments

View the payment history for each bill.
